# Modals

Screenshots of dialogs, confirmation prompts, slide-overs, sheets, and popovers.

## What to capture
- Modal sizing and positioning
- Overlay/backdrop treatment
- Header and close button styling
- Form-in-modal patterns
- Confirmation dialog patterns
- Slide-over/sheet from right edge
- Popover menus

## Recommended sources
- Linear: Issue detail modal, Confirmation dialogs
- Vercel: Deploy modal, Settings dialogs
- Stripe: Refund confirmation
- Notion: Page peek modal
