# Forms

Screenshots of input fields, select menus, date pickers, multi-step forms, and settings pages.

- Form field styling (inputs, textareas, selects)
- Settings page layouts
- Multi-step form flows
- Inline editing patterns
- Validation state styling (error, success)
- Date/time picker designs

- Linear: Issue creation form, Settings pages
- Stripe: Payment form, Dashboard settings
- Vercel: Project settings, Environment variables
