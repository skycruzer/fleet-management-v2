# Tables

Screenshots of data tables, list views, sortable columns, inline editing, and bulk action patterns.

## What to capture
- Table header styling
- Row density and spacing
- Sort/filter indicators
- Inline actions (edit, delete, status change)
- Bulk selection and batch actions
- Pagination patterns
- Responsive table behavior

## Recommended sources
- Linear: Issue list view
- Vercel: Deployment list, Log viewer
- Stripe: Payment/customer tables
- Notion: Database table views
