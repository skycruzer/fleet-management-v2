# Dashboards

Screenshots of dashboard layouts, metric cards, stat grids, analytics views, and overview pages.

## What to capture
- Metric card layouts (KPIs, sparklines, trend indicators)
- Dashboard grid arrangements
- Chart/graph styling and placement
- Data density patterns
- Header areas with filters and date pickers

## Recommended sources
- Linear: Project overview, My Issues dashboard
- Vercel: Deployment dashboard, Analytics
- Stripe: Revenue overview, Payments dashboard
- Notion: Team dashboard views
